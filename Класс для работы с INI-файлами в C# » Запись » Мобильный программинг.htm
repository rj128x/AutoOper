Класс для работы с INI-файлами в C#

18 Июнь 2009

Небольшой класс для чтения и записи данных в INI-файлы.
Скачать можно тут.

Создаем новый объект класса, в конструктор которого передаем путь к Вашему INI-файлу:
INIFile ini = new INIFile("C:\\test.ini");

Для чтения и записи данных использутся методы IniWriteValue и IniReadValue:

ini.IniWriteValue("Section","Key","Value");
string str = ini.IniReadValue("Section","Key");
ЗЫ: класс использует две WinAPI-шные фукнции из kernel32.dll, поэтому в Linux под mono работать не будет.
